Fidonet related Perl scripts

Nodelist
nl2sql: nl2sql.pl v1.x Standalone Perl script for loading the information into a fidonet/FTN nodelist MySQL database.
nl2db: nl2db.pl v0.x This version will use the (yet to be developed) FTN::Nodelist module to load the information in an FTN nodelist into a MySQL database.
ftnplnl - FTN::Nodelist - Perl module for reading FTN nodelists. (to be developed)
Packet
pkt2txt - pkt2txt.pl - Perl script to read FTN packets & output text files for the included messages
pkt2html - pkt2html.pl - Perl script to read FTN packets & output html files for the included messages
ftnplpkt - FTN::Packet - Perl module for reading/writing FTN packets. (to be developed)
Tick
ftnpltic - FTN::Tick - Perl module for processing FTN Tick files. (to be developed)
SRIF
srifpl - srif.pl - Perl module for processing SRIF files for file requests.
Utilities
ftnplutl - FTN::Utility - Perl module for miscellaneous FTN utility routines. (to be developed)
BBS
bbsdbpl - Perl scripts for an FTN compatible BBS, with the message base being held in a MySQL Database.
BBBS
mrtgbbbs - mrtgbbbs.pl - Perl script for reading the mrtg info from the BBBS standard port.
descadd - descadd.pl - Perl script to add File area descriptions from the FDN list into your BBBS filedirg.000 file.
indexer - indexer.pl - Perl script to generate web pages for bbbs file directories.
ticmsg - ticmsg.pl - Perl script to sort & organize file lists for posting.
Husky Project
hpt2sql - Perl scripts for using an SQL database with a Fidonet/FTN compatible Fidonet Tosser/Scanner, HPT from the Husky project. This version uses only the Perl hooks in HPT, standard Perl, & a MySQL database.
Maximus
af4max - Perl script which will produce a list of all files available for download from the file areas of a Maximus bulletin board system. Runs under Linux, DOS, OS/2 and Windows. (GNU GPL)

Fidonet Applications and Perl

The following is a sample listing of Fidonet related Software applications that can use or are already using Perl scripts.

Fidogate ( www.fidogate.org)
This is a Fidonet - Internet gateway application.
Has an extensive set of Perl scripts associated with it.
HPT ( husky.sourceforge.net/hpt.html)
Highly Portable Tosser; part of the Husky project.
Has an extensive set of Perl hooks built into it.
BBBS ( www.bbbs.net)
This is a BBS program that is available for many different platforms, and while it has it's own scripting language it can also be set up to use Perl.
