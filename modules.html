Fidonet related Perl Modules in the FTN namespace.

  • ftnpladx - FTN::Address - Perl module for processing FTN Addresses.
  • ftnplinb - FTN::Inbound - Perl module for processing FTN inbound directories.
  • ftnpljam - FTN::JAM - Perl module for FTN JAM Messagebase processing.
  • ftnpllog - FTN::Log - Perl module for FTN compatible Logging.
  • ftnplnl - FTN::Nodelist - Perl module for reading FTN nodelists.
  • ftnplout - FTN::Outbound - Perl module for processing FTN outbound directories.
  • ftnplpkt - FTN::Packet - Perl module for reading/writing FTN packets.
  • ftnplsrf - FTN::SRIF - Perl module for processing SRIF files for file requests.
  • ftnpltic - FTN::Tick - Perl module for processing FTN Tick files.
  • ftnplutl - FTN::Utility - Perl module for miscellaneous FTN utility routines.
