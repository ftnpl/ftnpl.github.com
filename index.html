Perl for Fidonet (FTN) Systems

Fidonet is an international amateur network of computer systems, often running public access bbs's, and all connected by modem (& often via the internet as well). This site is intended to provide a repository of Perl scripts, or links to where such Perl scripts might be found, for use on Fidonet Technology Networks (FTN) systems. A longer term goal is for the development of Perl Modules, using the FTN namespace.
